# Credit Scoring Model

## Objective
To predict whether a loan application should be approved using machine learning.

## Algorithms Used
 Logistic Regression
 Decision Tree
 Random Forest

## Libraries
 pandas
 numpy
 scikit-learn
 matplotlib

## Dataset Features
 Age
 Income
 LoanAmount
 CreditScore
 EmploymentYears

## Results
 Logistic Regression Accuracy: 88.33%
 Decision Tree Accuracy: 98.33%
-Random Forest Accuracy: 96.67%
